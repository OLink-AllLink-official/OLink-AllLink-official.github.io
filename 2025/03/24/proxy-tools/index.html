Soft Router/软路由

The soft router refers to a software-based router that uses regular computer hardware and software to perform routing tasks, as opposed to hardware-based routers which are dedicated devices designed specifically for network routing.

AOSP-based ROM/类原生

对于使用 Android 手机的用户，国产 OS 内置大量广告以及反诈骗 APP，十分影响体验，因此可以刷入基于 AOSP 制作的类原生 OS，没有广告且不臃肿，比较有名的有:

  • Pixel Experience https://get.pixelexperience.org/
  • Lineage OS https://lineageos.org/
  • Ares OS https://chriscatto.gitlab.io/AresOS/
  • CrDroid https://crdroid.net/

等等。鉴于国家政策，多数厂商关闭了 BL 解锁通道，目前仅有 OnePlus 用户可以轻松解锁 BootLoader 进行刷机操作，其他品牌均有各种问题，需要说明的是三星解锁 Bl 很简单，但是不能 root，有熔断机制。因此如果你有搞机需求，国产手机仅有 oneplus 还可以，国外品牌就很轻松了，Google Pixel，Samsung，Sony 均没有限制，其中 Google Pixel 的 ROM 最多。

Google Play APK

最新的国产Android系统内置Google服务，请注意，华为安卓鸿蒙系统(version ≤ HarmonyOS4.2)没有，但是可以使用其他开源项目来调用GMS，稍后会讲。在设置中有Google开关选项，唯有Google Play Store需要手动安装，安装包可以通过以下链接下载（无需魔法访问）：

  • TechSpot https://www.techspot.com/downloads/6105-google-play-store.html

MicroG

华为安卓鸿蒙系统(version ≤ HarmonyOS4.2)可以安装MicroG来替代GMS，不过华为手机无法安装Play Store或者说有办法安装，但是不能更新系统，一更新就不能用了，就得重新安装，并且安装非常麻烦，所以推荐大家使用MicroG+第三方App Store。MicroG作为开源程序，安全性可以得到保障，据说华为欧洲也在积极适配。

MicroG所需要的安装包项目主页均已提供，另外华为应用市场非国区也可下载：华为应用商店→我的→设置→国家/地区→选择的其他国家→输入账号密码验证→下载MicroG Companion和MicroG Server→打开手机设置-应用和服务-应用管理→打开MicroG服务APP详情页面→点击右上角齿轮→点击自我检查→添加Google账号，完成后即可使用。
